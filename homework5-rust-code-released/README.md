We strongly recommend that you use Visual Studio Code for this programming exercise.

## Sliding-Window Average Under a Hard Memory Constraint

You will implement an approximation algorithm for computing the sliding-window average under a memory constraint. We should be able to run this algorithm on an MCU with a limited memory budget. So, using more memory than specified is unacceptable.

Suggested steps:
1. Carefully read all the provided code.
2. Run all the tests that are provided in `src/wnd_exact.rs`.
3. Fill in your implementation in `src/wnd_apx.rs`.
4. Do NOT make any change to the fields of the struct `WndApx`.
5. Compile and run the executable whose source code is in `src/bin/main.rs`.
